Veritabanı Kurulumu — Supabase Bağlantısı
1
Supabase hesabı oluştur
supabase.com'a git, ücretsiz kayıt ol ve yeni bir proje oluştur.
2
SQL Editor'da tabloyu oluştur
Projen açıkken SQL Editor → New query → aşağıdaki SQL'i yapıştır ve çalıştır:

CREATE TABLE syp_data (id text PRIMARY KEY, user_id uuid REFERENCES auth.users NOT NULL, data jsonb, updated_at timestamptz DEFAULT now()); ALTER TABLE syp_data ENABLE ROW LEVEL SECURITY; CREATE POLICY "Users own data" ON syp_data FOR ALL USING (auth.uid() = user_id);
3
Project URL ve Anon Key'i al
Settings → API bölümünden Project URL ve anon public key'i kopyala.
